# 如何获取麦当劳MCP Token

## 官方文档链接
**https://open.mcd.cn/mcp/doc**

## 获取步骤

### 1. 访问官方文档
打开浏览器，访问上述链接，进入麦当劳MCP官方文档页面。

### 2. 了解MCP服务
在文档页面，您可以了解：
- 麦当劳MCP服务的基本介绍
- API功能和限制
- 使用条款和条件
- 技术支持信息

### 3. 注册和认证
根据文档指引：
1. **注册账号**：如果您还没有麦当劳开发者账号，需要先注册
2. **申请API访问**：提交API访问申请，说明使用目的
3. **等待审核**：麦当劳团队会审核您的申请
4. **获取Token**：审核通过后，您将获得个人MCP token

### 4. Token格式
麦当劳MCP token通常格式为：
```
Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```
或简单的字符串格式：
```
21lqF5yc7z5Pm9Im2TKOKMYSq3FYLcK4
```

### 5. 使用注意事项
1. **保密性**：Token相当于您的API密码，请妥善保管
2. **有效期**：Token可能有有效期，过期需要重新申请
3. **使用限制**：遵守API调用频率限制
4. **权限范围**：不同token可能有不同的API访问权限

## 常见问题

### Q: 为什么需要MCP token？
A: MCP token是访问麦当劳API的身份凭证，确保只有授权用户可以使用服务。

### Q: 获取token需要付费吗？
A: 根据麦当劳官方政策，基础API访问通常是免费的，但可能有使用限制。

### Q: token失效了怎么办？
A: 重新访问 **https://open.mcd.cn/mcp/doc**，按照指引重新申请或刷新token。

### Q: 可以分享token吗？
A: **不可以**。token是个人凭证，分享可能导致账户安全问题。

### Q: 在哪里查看token使用记录？
A: 在麦当劳开发者平台或联系技术支持查看API使用情况。

## 技术支持
如果在获取或使用token过程中遇到问题：
1. 仔细阅读官方文档
2. 查看常见问题解答
3. 联系麦当劳技术支持
4. 在开发者社区寻求帮助

## 安全建议
1. 不要在公共场合展示token
2. 不要在代码中硬编码token（使用环境变量）
3. 定期更换token
4. 监控API使用情况，发现异常及时处理

---

**重要提示**：本指南提供的是通用指引，具体流程可能因麦当劳政策调整而变化。请始终以官方文档 **https://open.mcd.cn/mcp/doc** 为准。
